二.区分函数和方法

借助types 模块判断是方法还是函数 from types import FunctionType, MethodType

  • 在类中:
    1. 类方法. 不论任何情况, 都是方法.
    2. 静态方法, 不论任何情况. 都是函数
    3. 实例方法, 如果是实例访问. 就是方法. 如果是类名访问就是函数

三. 反射

  • hasattr(obj, str) 判断对象中是否包含xxxx(str).

  • getattr(obj, str) 从对象中获取xxx(str)

    get表⽰示找, attr表⽰示属性(功能).

  • setattr(obj, str, value) 把对象中的str设置成value

  • delattr(obj, str) 从对象中删除xxxx(str)
